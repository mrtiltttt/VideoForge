  VideoForge — Як запустити на macOS
═══════════════════════════════════════════════════════

Опис: Автоматизоване створення відео (Script → Voice →
        Visuals → Video). GUI: CustomTkinter.

───────────────────────────────────────────────────────
  ПЕРЕДУМОВИ
───────────────────────────────────────────────────────

  • FFmpeg:   brew install ffmpeg

───────────────────────────────────────────────────────
  КРОК 1: Перейти в папку проекту
───────────────────────────────────────────────────────

  cd /шлях/до/VideoForge

───────────────────────────────────────────────────────
  КРОК 2: Створити віртуальне середовище (одноразово)
───────────────────────────────────────────────────────

  python3 -m venv venv
  source venv/bin/activate

───────────────────────────────────────────────────────
  КРОК 3: Встановити залежності (одноразово)
───────────────────────────────────────────────────────

  pip install "moviepy>=2.0.0" "Pillow>=10.0" "requests>=2.31" \
              "openai>=1.0" "pydub>=0.25" "numpy>=1.24" \
              "faster-whisper>=1.0" customtkinter

───────────────────────────────────────────────────────
  КРОК 4: Налаштувати API ключі
───────────────────────────────────────────────────────

  Скопіюй .env.example → .env та вкажи Pexels і OpenAI ключі:
  cp .env.example .env
  nano .env

───────────────────────────────────────────────────────
  КРОК 5: Запустити додаток
───────────────────────────────────────────────────────

  python3 gui.py

───────────────────────────────────────────────────────
  ШВИДКИЙ ЗАПУСК (якщо venv вже є)
───────────────────────────────────────────────────────

  source venv/bin/activate && python3 gui.py
